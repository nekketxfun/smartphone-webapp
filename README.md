# 🤖 Smartphone Catalog Telegram Bot

Современный Telegram бот с каталогом смартфонов и Web App интерфейсом.

## 🚀 Функционал

- 📱 Полный каталог смартфонов (5 брендов, 17 моделей)
- 🔍 Поиск по ценовым диапазонам
- 🏆 Топовые модели
- 📊 Статистика каталога
- 📱 Web App интерфейс
- 💾 SQLite база данных

## 🛠 Технологии

- Python 3.8+
- python-telegram-bot
- SQLite3
- HTML5, CSS3, JavaScript
- Telegram Web App SDK
